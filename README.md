# PJRCompiler

A compiler for PJRScript - a scripting language for [PJREngine](https://github.com/jegor377/PJREngine) - a text adventure game engine. It features the abilities like:

- dialogs,
- dialog options (a Finite State Machine),
- dialog images,
- background music,
- sound narration.

# Requirements

- Golang 1.25.4
